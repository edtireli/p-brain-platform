# p-brain Platform v1.0.0

Release date: 2026-01-09

## Highlights

- Desktop app (Tauri) bundling the web UI + local FastAPI bridge.
- QC/review UI for DCE-MRI and diffusion outputs (including tractography).
- Integration pointers to the core pipeline: https://github.com/edtireli/p-brain
- CNN bundle published on Zenodo: https://doi.org/10.5281/zenodo.15655348

## What’s included

- **Web UI (Vite/React)**: subject/projects browser, tractography viewer, QC overlays, pipeline status.
- **Desktop launcher (Tauri)**: ships the UI with a local bridge and Python environment bootstrap.
- **Backend bridge (FastAPI)**: local file access and serving tractography/AI outputs to the app.

## macOS download artifacts

Preferred:
- `p-brain_1.0.0_<arch>.dmg`

Optional fallback:
- `p-brain-macos.app.zip`

## Notes

- Desktop binaries should be distributed via GitHub Releases (not committed to git).
- If macOS Gatekeeper warns on first launch, right-click → Open, or remove quarantine after install:
  - `xattr -dr com.apple.quarantine /Applications/p-brain.app`
